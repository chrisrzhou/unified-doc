# Plugins

Apply valid [rehype](https://github.com/rehypejs/rehype/blob/master/doc/plugins.md) plugins to supercharge your document.

> Note that the `annotations` and `onSelectText` props may not behave correctly if rehype plugins modify the text content of the document since the callback is applied in relation to the source content.
