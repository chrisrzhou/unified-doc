# Styling

[View source](https://github.com/chrisrzhou/unified-doc/tree/master/docs/react-unified-doc/examples/styling.tsx)

Since `react-unified-doc` renders a non-opinionated and barebones HTML document that resembles the source content, you can easily style the document with regular CSS.  The renderer only creates `<mark />` tags with data attributes `data-start` and `data-end` to mark the start/end of annotations spanning multiple HTML elements.
